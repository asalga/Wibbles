# Wibbles by Andor
This is a clone of Nibbles

Levels are loaded from PNGs. Meta data for the levels can be found in scripts/boardMetaData.js

### Version
0.0.4

### Keys
 - Arrow keys to move
 - 'a' key to toggle audio
 - 'p' key to pause game
 - 'g' key to toggle god Mode
